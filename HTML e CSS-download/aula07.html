Nessa aula:

Existe a tag chamada <nav> que é navegação, onde se começa a usar as tag's para a criação de lista.

Existe a tag <ol> = lista ordenada; <ul> = lista não ordenada e <li> = item da lista, tbm se pode criar listas aninhadas(sublistas).

Tbm existem os tipos, que se usa com a tag <type>, que são '1','a'(A) e 'i'(I)
para as desordenadas são: 'circle', 'square' e 'disc'.

No caso das listas ordenadas se pode usar a tag <start> que vc pode usar números como índice indicando aonde irá começar.
