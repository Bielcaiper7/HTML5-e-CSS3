Eu posso utilizar a tag body na parte de head para configurar uma cor com style em CSS

As cores podem ser faladas pelo próprio programa que esta usando, ou você pode usar a chave de cor em hexadecimal, RGB, hsl (hue = matiz, saturation = saturação, lightness = luminosidade), etc.

Outra estilização tbm com alguns é colocando o 'a' de alpha no final onde se pode mudar a transparência das letras

E tbm se pode colocar imagens de fundo com o background-image: url, podendo pegar a URL de qualquer lugar
